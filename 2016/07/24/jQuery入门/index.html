jQuery入门

jQuery优势

轻量级、强大的选择器、DOM封装、可靠的事件处理机制、完善的Ajax、不污染顶级变量、出色的浏览器兼容性、链式操作方式等等。。。

jQuery 语法实例

$(this).hide() 隐藏当前的 HTML 元素。 
$("#test").hide() 隐藏 id="test" 的元素。元素选择器！ 
$("p").hide() 隐藏所有 <p> 元素。 CSS选择器。
$(".test").hide() 隐藏所有 class="test" 的元素。 类选择器！

jQuery 函数

所有 jQuery 函数位于一个 document ready 函数中。这是为了防止文档在完全加载（就绪）之前运行 jQuery 代码。如果在文档没有完全加载之前就运行函数，操作可能失败。1.试图隐藏一个不存在的元素 2.获得未完全加载的图像的大小 

window.onload=function(){};
$(document).ready(function(){});
传统的javaScript必须等页面中所有的内容都加载完毕(包括图片)才能执行function。
jQuery在页面中所有DOM结构绘制完毕之后就执行，可能DOM元素关联的东西还没有加载完。

jQuery对象和DOM对象。

DOM(Documents Object Model,文档对象模型)，每一份DOM都可以绘制成一棵树。
jQuery对象是jQuery对象就是通过jQuery包装DOM后产生的对象。
